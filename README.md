# 뉴스·게시물 링크 모니터

지정한 출처에서 키워드가 포함된 신규 링크를 로컬 화면에 모아 보여주는 개인용 모니터입니다.

## 실행

Windows에서 `start.bat`을 더블 클릭하거나 이 폴더에서 아래 명령을 실행합니다.

```powershell
python app.py
```

브라우저에서 `http://127.0.0.1:8765`가 열립니다. 별도 패키지 설치는 필요하지 않습니다.
macOS Python에 기본 인증서가 연결되지 않은 경우에는 운영체제 인증서 파일을 자동으로 사용합니다.

이미 8765 포트를 사용 중이면 실행 전에 `NEWS_LINK_MONITOR_PORT` 환경 변수로 다른 포트를 지정할 수 있습니다.

## 수집 방식

- 네이버 뉴스: 키워드별 최신 뉴스 검색 결과에서 제목이 일치하는 링크 수집
- 유튜브 채널: 공식 RSS 피드의 영상 중 제목에 키워드가 포함된 진행 중·예정 라이브만 수집

기본 확인 주기는 180초입니다. 화면에서 60~3600초 범위로 바꿀 수 있으며, 키워드와 출처도 화면에서 관리할 수 있습니다.

출처를 처음 확인하거나 URL·유형·키워드를 변경한 직후에는 기존 링크를 기준점으로만 저장합니다. 그 다음 확인부터 새로 발견된 링크가 화면에 표시됩니다. `data.json` 저장 전에는 기존 파일을 `data.json.bak`으로 백업합니다.

프로그램을 종료하면 화면에 표시되던 새 링크 목록이 `exports/news-links-YYYYMMDD-HHMMSS.csv` 파일로 저장됩니다. 저장할 새 링크가 없으면 CSV 파일은 만들지 않습니다.

브라우저 창이나 탭을 닫으면 약 15초 뒤 로컬 서버도 함께 종료됩니다. 새로고침처럼 곧바로 페이지가 다시 열리는 경우에는 종료하지 않습니다.
